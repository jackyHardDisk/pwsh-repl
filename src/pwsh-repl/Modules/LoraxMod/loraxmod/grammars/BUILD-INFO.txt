Build Notes:
------------
- All grammars use language version 15 (web-tree-sitter 0.25.x compatible)
- Compiled with -Os (optimize for size)
- SIDE_MODULE=2 for dynamic linking with web-tree-sitter
- All scanners use C (scanner.c), not C++ (scanner.cc)

To Rebuild:
-----------
See GRAMMAR-VERSIONS.md for detailed rebuild procedures and version tracking.

Prerequisites:
  Install emsdk to C:\tools\emsdk (Windows) or /usr/local/emsdk (Linux)
  git clone https://github.com/emscripten-core/emsdk.git
  ./emsdk install latest
  ./emsdk activate latest

Quick rebuild:
  1. Activate emsdk: powershell.exe -ExecutionPolicy Bypass -File C:\tools\emsdk\emsdk_env.ps1
  2. Run build script: bash build/build-grammar.sh
  3. Verify: node -e "require('./lib/index.js').getSupportedLanguages()"
